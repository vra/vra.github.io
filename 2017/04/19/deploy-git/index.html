搭建自己的Git服务器

发表于 2017-04-19 更新于 2021-08-20 阅读次数： 阅读次数：

相信很多人都对GitHub和GitLab很熟悉了，这些基于Git版本控制的在线代码托管平台由于丰富的内容，简洁的操作和集成一体化以及风靡全球了。今天我好奇，想了解下如何搭建自己的Git服务器，于是查了一些资料，记录下整个的流程。

为什么要用自己的Git服务器？

想了想，有下面的优势：

  1. 免费的私有仓库
  2. 完全的对项目的控制
  3. 了解GitHub和GitLab等背后的运作原理
